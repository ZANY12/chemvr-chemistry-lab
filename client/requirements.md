## Packages
three | Core 3D library
@react-three/fiber | React renderer for Three.js
@react-three/drei | Useful helpers for R3F
@react-three/xr | WebXR bindings for React
framer-motion | UI animations
leva | Debug GUI (optional but good for tweaking)
uuid | Unique IDs for objects

## Notes
The app requires WebXR compatible browser or emulator for full VR features.
Standard '2D' mode works on desktop via OrbitControls.
Textures/models are procedural to keep it lightweight.
